─ Portfolio ─

Some projects I've worked on

Omnifood

Website made to practice conventional front-end fundamentals including design techniques and responsiveness. Site features an imaginative AI-aided meal plan subscription service.

Sentence Game

Multiplayer game site where users can choose a screen name, join a private room and play a simple-to-learn game with others in real time to build silly sentences. At the end, players get to share completed sentences and generated images.

Mapty

Site to track running or cycling workouts by placing pins on an interactive and live map based on the user's location. Recorded workouts will be stored in an easy-to-view list for the user's history and convenience.

Forkify

A site for finding recipes easily. Users can input ingredients into a search bar and associated recipes will be loaded for the user to browse. Users can bookmark recipes and also adjust portion size.
